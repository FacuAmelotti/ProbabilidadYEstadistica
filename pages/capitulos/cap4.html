Modelos de Probabilidad Continua

En esta unidad se estudian las distribuciones de probabilidad continua, donde las variables aleatorias pueden tomar un número infinito de valores posibles dentro de un intervalo. Se analizan funciones de densidad, cálculo de probabilidades mediante integración, y las distribuciones Uniforme y Exponencial como casos representativos, junto con su relación con la distribución de Poisson.

📏Distribución Uniforme Continua

Una variable aleatoria continua \( X \) tiene distribución uniforme en el intervalo \([a,b]\), con \( a < b \), si su función de densidad de probabilidad (fdp) es:

$$f(x) = \begin{cases} \frac{1}{b - a}, & a \le x \le b \\ 0, & \text{caso contrario} \end{cases}$$

Se denota como:

$$X \sim U(a,b)$$

Propiedades

  • \( f(x) \ge 0 \) para todo \( x \).
  • \( \int_{-\infty}^{\infty} f(x)\,dx = 1 \).

Verificación:

$$\int_a^b \frac{1}{b-a} dx = \frac{x}{b-a} \bigg|_a^b = \frac{b-a}{b-a} = 1$$
Ejemplo — Espera del colectivo

Los colectivos de una línea llegan cada 15 minutos, comenzando a las 7:00 am. Si un pasajero llega en cualquier momento entre las 7:00 y las 7:30, se tiene:

$$X \sim U(0, 30)$$
  • Probabilidad de esperar menos de 5 minutos: \( P(10 \le X \le 15) + P(25 \le X \le 30) \)
  • Probabilidad de esperar más de 10 minutos: \( P(0 \le X \le 5) + P(15 \le X \le 20) \)
$$P = \frac{(15 - 10) + (30 - 25)}{30 - 0} = \frac{10}{30} = \frac{1}{3}$$

Esperanza y Varianza

$$E[X] = \frac{a + b}{2}, \quad V[X] = \frac{(b - a)^2}{12}$$

Demostración simplificada de la varianza

$$E[X] = \int_a^b \frac{x}{b - a} dx = \frac{b^2 - a^2}{2(b - a)} = \frac{a + b}{2}$$
$$E[X^2] = \int_a^b \frac{x^2}{b - a} dx = \frac{b^3 - a^3}{3(b - a)}$$

Entonces: $$V[X] = E[X^2] - [E[X]]^2 = \frac{(b - a)^2}{12}$$

⏱️Distribución Exponencial

Una variable aleatoria continua \( X \) tiene distribución exponencial con parámetro \( \lambda > 0 \) si su función de densidad de probabilidad es:

$$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x \ge 0 \\ 0, & \text{caso contrario} \end{cases}$$

Se denota como:

$$X \sim E(\lambda)$$

Propiedades

$$E[X] = \frac{1}{\lambda}, \quad V[X] = \frac{1}{\lambda^2}$$

Modela tiempos de espera o duración de eventos hasta la primera ocurrencia.

Ejemplo — Tiempo de respuesta

Supongamos que el tiempo de respuesta (en segundos) de una terminal sigue una distribución \( X \sim E(0.2) \).

  • \( P(X \le 10) = 1 - e^{-0.2(10)} = 1 - e^{-2} = 0.86 \)
  • \( P(5 \le X \le 10) = e^{-1} - e^{-2} = 0.23 \)

🔗Relación entre Distribución Exponencial y Poisson

Si los eventos siguen un proceso de Poisson con parámetro \( \lambda \), y definimos \( T \) como el tiempo de espera hasta el siguiente evento, entonces:

$$P(T > t) = P(X = 0) = e^{-\lambda t}$$

Por tanto:

$$P(T \le t) = 1 - e^{-\lambda t}$$

Esto corresponde a la función de distribución acumulada (FDA) de una variable exponencial. Es decir, \( T \sim E(\lambda) \).

Ejemplo — Inundaciones

Sea \( X \): número de inundaciones en un periodo \([0, t]\) con \( X \sim P(\lambda t) \).

  • El número medio de inundaciones en ese tiempo es \( E[X] = \lambda t \).
  • El tiempo entre inundaciones se modela como \( T \sim E(\lambda) \).
  • El tiempo medio de retorno es \( E[T] = 1 / \lambda \).

📈Distribución Normal (Gaussiana)

Sea \( X \) una variable aleatoria continua. Decimos que tiene distribución normal con parámetros \( \mu \) y \( \sigma \) si su función de densidad de probabilidad es:

$$f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x - \mu}{\sigma}\right)^2}, \quad \mu \in \mathbb{R}, \ \sigma > 0$$

Observaciones

  • El gráfico es simétrico alrededor de \( \mu \).
  • El parámetro \( \mu \) representa la posición (traslada la curva).
  • El parámetro \( \sigma \) controla la escala o ancho de la curva.

Se denota como:

$$X \sim N(\mu, \sigma^2)$$

Esperanza y Varianza

$$E[X] = \mu, \quad V[X] = \sigma^2$$

Distribución Normal Estándar

Si \( \mu = 0 \) y \( \sigma = 1 \), se dice que \( X \) sigue una distribución normal estándar:

$$X \sim N(0,1)$$

Su función de densidad y distribución acumulada son:

$$\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2 / 2}, \qquad \Phi(x) = P(X \le x) = \int_{-\infty}^{x} \varphi(t) \, dt$$

La integral de \( \Phi(x) \) no puede expresarse en términos de funciones elementales. Por ello, se utilizan tablas de valores de \( \Phi(x) \) para aproximar probabilidades.

Propiedades útiles

  • \( \Phi(-x) = 1 - \Phi(x) \)
  • Para \( x < -4 \Rightarrow \Phi(x) \approx 0 \)
  • Para \( x > 4 \Rightarrow \Phi(x) \approx 1 \)

Ejemplos con tabla de la Normal Estándar

Ejemplo 1

\( P(X \le 1.26) = \Phi(1.26) = 0.89616 \)

\( P(X > 1.26) = 1 - \Phi(1.26) = 0.10384 \)

\( P(X > -1.37) = P(X \le 1.37) = \Phi(1.37) = 0.91465 \)

\( P(-1.25 < X < 0.37) = \Phi(0.37) - \Phi(-1.25) = \Phi(0.37) - [1 - \Phi(1.25)] = 0.53866 \)

Ejemplo 2

Para qué valor de \( x \) se cumple \( P(-x < X < x) = 0.95 \):

$$2\Phi(x) - 1 = 0.95 \Rightarrow \Phi(x) = 0.975 \Rightarrow x = 1.96$$

Transformación Lineal y Estandarización

Si \( X \sim N(\mu, \sigma^2) \) y \( Y = aX + b \), entonces:

$$Y \sim N(a\mu + b, a^2\sigma^2)$$

En particular, al definir:

$$Z = \frac{X - \mu}{\sigma} \Rightarrow Z \sim N(0,1)$$

Se tiene entonces la relación:

$$P(X \le x) = P\left(Z \le \frac{x - \mu}{\sigma}\right) = \Phi\left(\frac{x - \mu}{\sigma}\right)$$
Ejemplo 3 — Normalización

Si \( X \sim N(3, 9) \), es decir \( \mu = 3, \sigma = 3 \):

  • \( P(2 < X < 5) = \Phi(2/3) - \Phi(-1/3) = \Phi(2/3) - [1 - \Phi(1/3)] = 0.3779 \)
  • \( P(X > 0) = \Phi(1) = 0.8413 \)
  • \( P(|X - 3| > 6) = 2[1 - \Phi(2)] = 0.0456 \)
Ejemplo 4 — Máquinas de corchos

Se comparan dos máquinas que producen corchos con diámetros normales:

  • \( X \sim N(3, 0.1^2) \)
  • \( Y \sim N(3.04, 0.02^2) \)

Corchos aceptables: \( 2.9 \le d \le 3.1 \)

Para la máquina 1: \( P(2.9 \le X \le 3.1) = 2\Phi(1) - 1 = 0.683 \)

Para la máquina 2: \( P(2.9 \le Y \le 3.1) = \Phi(3) - \Phi(-7) = \Phi(3) = 0.99 \)

Conclusión: la máquina 2 tiene mayor probabilidad de producir corchos aceptables.

Regla práctica

Cualquier situación donde intervengan muchos factores aleatorios independientes tiende a seguir una distribución normal.

Ejemplo: diferencias entre valores medidos o muestreados de una misma población.

📊Distribución Chi-Cuadrado (χ²)

Recordemos que \( S^2 \) es la varianza muestral, la cual se usa para estimar la varianza poblacional \( \sigma^2 \) de una distribución normal. Se define como:

$$S^2 = \frac{\sum_{i=1}^{n}(x_i - \overline{X})^2}{n - 1}$$

Para evaluar qué tan buena es esta estimación de \( \sigma^2 \), se utiliza el estadístico Chi-Cuadrado (χ²):

$$\chi^2 = \frac{(n - 1) S^2}{\sigma^2}, \quad \nu = n - 1 \text{ (grados de libertad)}$$

Observaciones

  • Los valores posibles de \( \chi^2 \) son siempre \( \ge 0 \).
  • La distribución depende del número de grados de libertad \( \nu \).
  • No es simétrica; tiene una forma sesgada hacia la derecha.

La variable \( \chi^2 \) puede considerarse una versión estandarizada de la varianza muestral. Si los datos provienen de una población normal:

$$X \sim N(\mu, \sigma^2) \Rightarrow Z \sim N(0, 1)$$

Ejemplo

Sea una muestra aleatoria de tamaño \( n = 25 \) proveniente de una población normal con \( \sigma^2 = 6 \). Se desea calcular la probabilidad de que la varianza muestral sea:

  • Mayor que \( 9.1 \)
  • Entre \( 3.462 \) y \( 10.745 \)
\[ P(S^2 > 9.1) = P\left(\chi^2 > \frac{(25 - 1)(9.1)}{6}\right) = P(\chi^2 > 36.4) = 1 - P(\chi^2 \le 36.4) = 0.05 \]
\[ P(3.462 \le S^2 \le 10.745) = P(13.84 \le \chi^2 \le 42.98) = P(\chi^2 \le 42.98) - P(\chi^2 \le 13.84) = 0.95 - 0.01 = 0.94 \]

📉Distribución t de Student

Cuando se estima la media poblacional \( \mu \) a partir de una muestra pequeña y la varianza \( \sigma^2 \) es desconocida, se utiliza la distribución t de Student.

Recordemos que la media muestral se define como:

$$\overline{X} = \frac{\sum_{i=1}^{n} x_i}{n}$$

Si \( \sigma \) es conocida, entonces:

$$Z = \frac{\overline{X} - \mu}{\sigma / \sqrt{n}}$$

Pero si \( \sigma \) es desconocida, se reemplaza por \( S \) y se obtiene el estadístico t:

$$T = \frac{\overline{X} - \mu}{S / \sqrt{n}}$$

Observaciones

  • Tiene forma de campana, centrada en el origen.
  • Es más ancha que la distribución normal estándar.
  • Al aumentar los grados de libertad \( \nu = n - 1 \), se asemeja más a \( N(0,1) \).

Ejemplo

Calcular \( k \) tal que \( P(k < T < -1.761) = 0.045 \) para una muestra de tamaño \( n = 15 \) (grados de libertad \( \nu = 14 \)).

\[ t_{0.05,14} = 1.761 \Rightarrow -t_{0.05} = -1.761 \] \[ 0.045 = 0.05 - \alpha \Rightarrow \alpha = 0.005 \] \[ k = -t_{0.005,14} = -2.977 \]

Por tanto: \( P(-2.977 < T < 2.977) = 0.045 \)

Conclusión

La distribución t-Student se utiliza para inferencias sobre medias con varianza desconocida y tamaños muestrales pequeños.

📋Resumen General

  • Las variables aleatorias representan numéricamente los resultados de un experimento.
  • Las discretas toman valores finitos o contables, y se describen mediante una función de probabilidad.
  • La función de distribución acumulada resume la probabilidad acumulada hasta un punto.
  • La media y varianza describen el comportamiento promedio y la dispersión.
  • Modelos discretos importantes: Bernoulli, Binomial, Poisson, Geométrica, Hipergeométrica.
