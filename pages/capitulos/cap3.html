üéØDistribuci√≥n Geom√©trica

La distribuci√≥n geom√©trica describe el n√∫mero de repeticiones de un experimento aleatorio necesarias hasta obtener el primer √©xito. Es un caso particular de la distribuci√≥n binomial, donde se repite el experimento hasta el primer evento favorable.

$$P(X = k) = p(1 - p)^{k - 1}, \quad k = 1, 2, 3, \dots$$

Ejemplo ‚Äî Falla de una computadora

$$P(X = 12) = 0.1 (1 - 0.1)^{12 - 1} = 0.1 (0.9)^{11} = 0.031$$

$$\sum_{k=1}^{\infty} P(X = k) = \sum_{k=1}^{\infty} p(1 - p)^{k - 1} = p \sum_{k=0}^{\infty} (1 - p)^k = p \frac{1}{1 - (1 - p)} = 1$$

$$E[X] = \frac{1}{p}, \quad V[X] = \frac{1 - p}{p^2}$$
