Modelos de Probabilidad Continua

En esta unidad se estudian las distribuciones de probabilidad continua, donde las variables aleatorias pueden tomar un n√∫mero infinito de valores posibles dentro de un intervalo. Se analizan funciones de densidad, c√°lculo de probabilidades mediante integraci√≥n, y las distribuciones Uniforme y Exponencial como casos representativos, junto con su relaci√≥n con la distribuci√≥n de Poisson.

üìèDistribuci√≥n Uniforme Continua

Una variable aleatoria continua \( X \) tiene distribuci√≥n uniforme en el intervalo \([a,b]\), con \( a < b \), si su funci√≥n de densidad de probabilidad (fdp) es:

$$f(x) = \begin{cases} \frac{1}{b - a}, & a \le x \le b \\ 0, & \text{caso contrario} \end{cases}$$

Se denota como:

$$X \sim U(a,b)$$

Propiedades

  • \( f(x) \ge 0 \) para todo \( x \).
  • \( \int_{-\infty}^{\infty} f(x)\,dx = 1 \).

Verificaci√≥n:

$$\int_a^b \frac{1}{b-a} dx = \frac{x}{b-a} \bigg|_a^b = \frac{b-a}{b-a} = 1$$
Ejemplo ‚Äî Espera del colectivo

Los colectivos de una l√≠nea llegan cada 15 minutos, comenzando a las 7:00 am. Si un pasajero llega en cualquier momento entre las 7:00 y las 7:30, se tiene:

$$X \sim U(0, 30)$$
  • Probabilidad de esperar menos de 5 minutos: \( P(10 \le X \le 15) + P(25 \le X \le 30) \)
  • Probabilidad de esperar m√°s de 10 minutos: \( P(0 \le X \le 5) + P(15 \le X \le 20) \)
$$P = \frac{(15 - 10) + (30 - 25)}{30 - 0} = \frac{10}{30} = \frac{1}{3}$$

Esperanza y Varianza

$$E[X] = \frac{a + b}{2}, \quad V[X] = \frac{(b - a)^2}{12}$$

Demostraci√≥n simplificada de la varianza

$$E[X] = \int_a^b \frac{x}{b - a} dx = \frac{b^2 - a^2}{2(b - a)} = \frac{a + b}{2}$$
$$E[X^2] = \int_a^b \frac{x^2}{b - a} dx = \frac{b^3 - a^3}{3(b - a)}$$

Entonces: $$V[X] = E[X^2] - [E[X]]^2 = \frac{(b - a)^2}{12}$$

‚è±Ô∏èDistribuci√≥n Exponencial

Una variable aleatoria continua \( X \) tiene distribuci√≥n exponencial con par√°metro \( \lambda > 0 \) si su funci√≥n de densidad de probabilidad es:

$$f(x) = \begin{cases} \lambda e^{-\lambda x}, & x \ge 0 \\ 0, & \text{caso contrario} \end{cases}$$

Se denota como:

$$X \sim E(\lambda)$$

Propiedades

$$E[X] = \frac{1}{\lambda}, \quad V[X] = \frac{1}{\lambda^2}$$

Modela tiempos de espera o duraci√≥n de eventos hasta la primera ocurrencia.

Ejemplo ‚Äî Tiempo de respuesta

Supongamos que el tiempo de respuesta (en segundos) de una terminal sigue una distribuci√≥n \( X \sim E(0.2) \).

  • \( P(X \le 10) = 1 - e^{-0.2(10)} = 1 - e^{-2} = 0.86 \)
  • \( P(5 \le X \le 10) = e^{-1} - e^{-2} = 0.23 \)

üîóRelaci√≥n entre Distribuci√≥n Exponencial y Poisson

Si los eventos siguen un proceso de Poisson con par√°metro \( \lambda \), y definimos \( T \) como el tiempo de espera hasta el siguiente evento, entonces:

$$P(T > t) = P(X = 0) = e^{-\lambda t}$$

Por tanto:

$$P(T \le t) = 1 - e^{-\lambda t}$$

Esto corresponde a la funci√≥n de distribuci√≥n acumulada (FDA) de una variable exponencial. Es decir, \( T \sim E(\lambda) \).

Ejemplo ‚Äî Inundaciones

Sea \( X \): n√∫mero de inundaciones en un periodo \([0, t]\) con \( X \sim P(\lambda t) \).

  • El n√∫mero medio de inundaciones en ese tiempo es \( E[X] = \lambda t \).
  • El tiempo entre inundaciones se modela como \( T \sim E(\lambda) \).
  • El tiempo medio de retorno es \( E[T] = 1 / \lambda \).

üìàDistribuci√≥n Normal (Gaussiana)

Sea \( X \) una variable aleatoria continua. Decimos que tiene distribuci√≥n normal con par√°metros \( \mu \) y \( \sigma \) si su funci√≥n de densidad de probabilidad es:

$$f(x) = \frac{1}{\sigma \sqrt{2\pi}} e^{-\frac{1}{2}\left(\frac{x - \mu}{\sigma}\right)^2}, \quad \mu \in \mathbb{R}, \ \sigma > 0$$

  • El par√°metro \( \sigma \) controla la escala o ancho de la curva.

Se denota como:

$$X \sim N(\mu, \sigma^2)$$

$$E[X] = \mu, \quad V[X] = \sigma^2$$

Distribuci√≥n Normal Est√°ndar

Si \( \mu = 0 \) y \( \sigma = 1 \), se dice que \( X \) sigue una distribuci√≥n normal est√°ndar:

$$X \sim N(0,1)$$

Su funci√≥n de densidad y distribuci√≥n acumulada son:

$$\varphi(x) = \frac{1}{\sqrt{2\pi}} e^{-x^2 / 2}, \qquad \Phi(x) = P(X \le x) = \int_{-\infty}^{x} \varphi(t) \, dt$$

La integral de \( \Phi(x) \) no puede expresarse en t√©rminos de funciones elementales. Por ello, se utilizan tablas de valores de \( \Phi(x) \) para aproximar probabilidades.

Propiedades √∫tiles

  • \( \Phi(-x) = 1 - \Phi(x) \)
  • Para \( x < -4 \Rightarrow \Phi(x) \approx 0 \)
  • Para \( x > 4 \Rightarrow \Phi(x) \approx 1 \)

Ejemplos con tabla de la Normal Est√°ndar

Ejemplo 1

\( P(X \le 1.26) = \Phi(1.26) = 0.89616 \)

\( P(X > 1.26) = 1 - \Phi(1.26) = 0.10384 \)

\( P(X > -1.37) = P(X \le 1.37) = \Phi(1.37) = 0.91465 \)

\( P(-1.25 < X < 0.37) = \Phi(0.37) - \Phi(-1.25) = \Phi(0.37) - [1 - \Phi(1.25)] = 0.53866 \)

Ejemplo 2

Para qu√© valor de \( x \) se cumple \( P(-x < X < x) = 0.95 \):

$$2\Phi(x) - 1 = 0.95 \Rightarrow \Phi(x) = 0.975 \Rightarrow x = 1.96$$

Transformaci√≥n Lineal y Estandarizaci√≥n

Si \( X \sim N(\mu, \sigma^2) \) y \( Y = aX + b \), entonces:

$$Y \sim N(a\mu + b, a^2\sigma^2)$$

En particular, al definir:

$$Z = \frac{X - \mu}{\sigma} \Rightarrow Z \sim N(0,1)$$

Se tiene entonces la relaci√≥n:

$$P(X \le x) = P\left(Z \le \frac{x - \mu}{\sigma}\right) = \Phi\left(\frac{x - \mu}{\sigma}\right)$$
Ejemplo 3 ‚Äî Normalizaci√≥n

Si \( X \sim N(3, 9) \), es decir \( \mu = 3, \sigma = 3 \):

  • \( P(2 < X < 5) = \Phi(2/3) - \Phi(-1/3) = \Phi(2/3) - [1 - \Phi(1/3)] = 0.3779 \)
  • \( P(X > 0) = \Phi(1) = 0.8413 \)
  • \( P(|X - 3| > 6) = 2[1 - \Phi(2)] = 0.0456 \)
Ejemplo 4 ‚Äî M√°quinas de corchos

Se comparan dos m√°quinas que producen corchos con di√°metros normales:

  • \( X \sim N(3, 0.1^2) \)

üìäDistribuci√≥n Chi-Cuadrado (œá¬≤)

$$S^2 = \frac{\sum_{i=1}^{n}(x_i - \overline{X})^2}{n - 1}$$

$$X \sim N(\mu, \sigma^2) \Rightarrow Z \sim N(0, 1)$$

  • Mayor que \( 9.1 \)

\[ P(S^2 > 9.1) = P\left(\chi^2 > \frac{(25 - 1)(9.1)}{6}\right) = P(\chi^2 > 36.4) = 1 - P(\chi^2 \le 36.4) = 0.05 \]
\[ P(3.462 \le S^2 \le 10.745) = P(13.84 \le \chi^2 \le 42.98) = P(\chi^2 \le 42.98) - P(\chi^2 \le 13.84) = 0.95 - 0.01 = 0.94 \]

üìâDistribuci√≥n t de Student

Cuando se estima la media poblacional \( \mu \) a partir de una muestra peque√±a y la varianza \( \sigma^2 \) es desconocida, se utiliza la distribuci√≥n t de Student.

$$\overline{X} = \frac{\sum_{i=1}^{n} x_i}{n}$$

Si \( \sigma \) es conocida, entonces:

$$Z = \frac{\overline{X} - \mu}{\sigma / \sqrt{n}}$$

Pero si \( \sigma \) es desconocida, se reemplaza por \( S \) y se obtiene el estad√≠stico t:

$$T = \frac{\overline{X} - \mu}{S / \sqrt{n}}$$

  • Es m√°s ancha que la distribuci√≥n normal est√°ndar.
  • Al aumentar los grados de libertad \( \nu = n - 1 \), se asemeja m√°s a \( N(0,1) \).

Ejemplo

\[ t_{0.05,14} = 1.761 \Rightarrow -t_{0.05} = -1.761 \] \[ 0.045 = 0.05 - \alpha \Rightarrow \alpha = 0.005 \] \[ k = -t_{0.005,14} = -2.977 \]

Por tanto: \( P(-2.977 < T < 2.977) = 0.045 \)

Conclusi√≥n

La distribuci√≥n t-Student se utiliza para inferencias sobre medias con varianza desconocida y tama√±os muestrales peque√±os.
